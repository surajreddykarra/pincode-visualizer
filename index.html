Competition details

Same dataset, two different implementations.

Shared source material

Each agent started from the same source CSV containing Indian PIN codes, location information, and related metadata.

Independent implementations

The folders prefixed with cc- were produced by Claude Code, and the folders prefixed with codex- were produced by Codex.

Precomputed outputs reused

This site uses the precomputed data already generated by each implementation so the public deployment stays static and fast.

No raw CSV in production bundle

The deployment is organized to ship only the runtime assets needed by the two demos, not the source CSV or preprocessing scripts.
